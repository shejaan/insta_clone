# Instagram Clone (Django)

A simple Instagram-like social media application built using Django.

## Features

- User registration and login
- Upload posts
- Like and comment system
- User profiles
- Follow system
- Messaging feature

## Tech Stack

Backend:
- Django
- Python

Frontend:
- HTML
- CSS
- JavaScript

Database:
- SQLite

## Installation

Clone repository

git clone https://github.com/YOUR_USERNAME/insta_clone.git

Go to project folder

cd insta_clone

Create virtual environment

python -m venv venv

Activate environment

pip install -r requirements.txt

Run migrations

python manage.py migrate

Run server

python manage.py runserver
